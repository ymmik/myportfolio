Dashboard Wireflow Design
property management with smart lock

User flow / Wireframe Design

This service aims at providing property managers a more convenient way to manage their property with igloohome smart lock. For this we proposed a property management dashboard design.

User Story

User stories for Property Managers:
As a property manager, I have to manage over 100s of properties at once for my company.

"As a Property Manager I want to ..."

Goal 1: User wants to have a quick check on main issue
Proposed feature:
timetable or list of ongoing cases, showing property name, location, entry time and purpose

Goal 2: User wants to check out a property for details
Proposed feature:
(1) search bar > search by entering keywords
(2) categorize by folders/tags

Goal 3: User wants to grant access to house viewer
Proposed feature:
generate one time entry PIN(s) and send to viewer

Goal 4: User wants to grant access to cleaner / maintainer
Proposed feature:
generate temporary PIN and send to cleaner/maintainer

Goal 5: User want to check the access log
Proposed feature:
list in chronological order

Goal 8: Check if client enter/leave the property as scheduled
Proposed feature:
(1)Access log > see last entry time(make sure user is using it as scheduled); set up validation of PIN so that the viewer has to leave by the validation anyway.
(2)give last entry and leaving time of the property

Goal 7: Add /delete/ edit my properties info from my listing
Proposed feature:
Editable list
